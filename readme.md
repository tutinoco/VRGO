# 準備
1. VRChat Creator Companionを起動する。
2. Project → New → UdonSharpを選択し、適当な場所に「VRGO」のプロジェクト名を設定してCreateを押す。
3. Terminalでプロジェクトフォルダに移動して次のコマンドを実行する。
```
	git init .;
	git remote add origin https://github.com/tutinoco/VRGO.git;
	git pull origin main;
	git submodule update --init --recursive;
```
